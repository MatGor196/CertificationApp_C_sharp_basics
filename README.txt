# CertificationApp_C_sharp_basics
Repository with simple planner.

Prosta lista zadań, która na każdy z siedmiu dni tygodnia umożliwa dodanie osobnej listy rzeczy do wykonania. Do każdego zadania będzie można dopisać godzinę (liczba całkowita 0 - 23) wykonania i priorytet (w skali od 1-5). W aplikacji będzie można wyświetlić listy rzeczy do zrobienia, albo ze względu na priorytet (priorytetami malejąco od góry), albo ze względu na godzinę wykonania (godzinami rosnąco w dół).
